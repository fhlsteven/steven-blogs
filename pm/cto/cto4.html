# 你与“CXO”们的差距有多远？

高管，对白领阶层是个引无数英雄竞折腰的称号，就象每个战士都想做将军一样，三分之二的白领都想做企业的高管。然而，一个企业只有几个高管，金字塔尖上只有几颗闪烁着神秘而迷人光芒的夜明珠。令白领们心驰神往、梦寐以求。

那么现在你距高管的差距有多大，是尚未点开死穴，还是初入上升通道？是顺风顺水，平云直上，还是虽然只有一步之遥，却隔着玻璃天花板，可望不可及？今生能否登上金字塔，去摘取这夜明珠的美梦能否成真？叱咤风云的高管需要具备哪些能力？职场白领又如何历练自我奔高管？宏威职业顾问将为你指点迷津、点开死穴，使你进入高管上升通道。

# 一、 各类高管的特质

  1. 高级管理人才CEO这个概念，近年来被炒的火热。在国外把CEO称为总监、总裁、董事长、企业精英、商业领袖以及经营管理的首席或最高执行官等，CEO是企业的核心与中坚人物，其道德修养和才识学问及洞察力、影响力、危机处理能力好坏直接影响到企业的前途与命运。

  2. 高级财务管理人才CFO CFO是企业首席财务官的英文缩写，财务管理的重要性不言而喻，上至一个国家，下至每一个人都应做好财务管理工作。CFO除了要具备高深的财务方面的和钱财智商外，还要掌握逻辑推理能力、语言与数字推理能力等多方面的知识及财运目标、自信心远大抱负。

  3. 高级技术人才CTO CTO是企业生存和发展所必不可少的重要因素，而良好的空间想象力和独特的思维方式，又是CTO所必须具备的。

  4. 高级行政人才COO COO应具备的是专业知识、信念合理、坚忍不拔、思维能力、稳定性、操作灵活性能够及时掌握团队工作方式适应市场快速变化的原因、采用团队工作方式的深层原因。

  5. 高级信息管理人才CIO CIO在飞跃发展的信息时代，能够使企业通过不断发展与改进的信息技术，不断提高客户服务水平，降低运营成本。CIO是企业与消费者之间沟通的桥梁，企业只有能够正确掌握消费者的需求信息，才能在激烈的竞争浪潮中立于不败之地。CIO除了具备专业能力、图形文字处理能力统筹应变能力、直觉应变能力外还要具备竞争意识、创新意识和思维敏感度。

  6. 市场营销人才CMO CMO的能力是一个企业、组织能否赢得市场的关键，除专业能力、推销能力、经营管理能力外还要具备目标实现能力和信心、应变能力、交流沟通能力及健康的体魄和意志力，才能在工商界施展才华，为企业带来不菲的效益。

  7. 人力资源管理人才CHO CHO企业与员工沟通的桥梁，要具备与上司沟通的能力和很强的识人、培训能力、对员工的工作能力有很强的识别度，更重要的是有一种特质的亲和力、性格活泼开朗、对人对事热心而富于感情。

此外，还有常务副总裁（EVPS）、高级副总裁判（SVP）可参考上述。

# 二、 走向高管的上升通道

  1. 很多企业的老总无一例外地认为，真正意义上的高管，其对企业的忠诚度和自身业务能力必须经受时间的考验，考验时间一般长达2年左右。同时企业老总都认为那些随同企业一起成长或者中途加入企业、经过选拔和培养的人才最适合做高管。为了便于选拔，企业制定了严格的绩效考核标准和经常提供员工换岗工作的机会，借以更好地发现人才。

  2. 有企业家精神、富有战略规划头脑和出色的战略思维能力、正确指明公司发展方向的人才是企业真正需要的高管的不二人选，持这一观点的人占61%.高管能否带给企业清晰的发展思路最为重要；其次是优越的执行能力。

    本次调查中。有85%的白领坚信，卓越的领导能力可以经过后天训练而成；只有15%的人认为领导力是天赐的礼物，将军是天生的。这就是为什么那么多职场人士希望达到高管职位的原因，他们相信事在人为，经过一番努力，自己一定能有非凡的作为。大多数人自我历练奔高管

  3. 不断为自己充电，有部分白领在优厚的生活和沉重的工作压力下，已渐渐失去了学习动力，作为高管，不断学习新的知识，掌握行业内最先进的技术和理念，有跨文化的敏感能力，理解不同文化背景下的不同思维方式和工作方式，才有可能作为领头人。39%的人选择参加专业的学历培训管理类的短训等学历和能力的学习：25%的人计划加强人际交往能力，搭建关系网络来实现晋升之路。在流行高学历和越来越重视人际交往，强调人脉即财脉的今天，这两种方式不失为成熟的做法。相反，6%的白领打算通过阅读名人传记、励志图书等来实现高管之梦，这未免显得学生气了。

  4. 积累的丰厚经验，观摩其他企业成功或失败的发展轨迹，最好能争取在其他企业工作的机会，从基层岗位做起，充分体会创业的艰难。对企业的每一个部门、工作环节，都有具体的了解，深刻体会基层员工的工作感受。经历企业内部不同的岗位尤其是营销、财务人力资源三个关键部门。担任能迅速产生业绩的职务，如销售、企业战略研究等，通过努力，迅速看到所产生的效应。这可以最大化激励和磨练个人意志，也为建立个人威信打下基础。

  5. 有良好IQ（智商）更要有良好的情商（EQ）。懂得领导的艺术，广结人脉，有精英团队做自己的得力臂膀。在不同发展时期了解员工想要的是什么，培养识人用人的能力，的如果是家族企业，企业的所有权和经营权应该是分开的，聘请职业经理人来运营，家族拥有制约机制。

  6. 良好的职业道德和个人特质，堂堂正正、光明磊落。一颗能容纳百川的胸怀，善于听取、关心、理解、预知他人心理和情感的变化，不论在业内还是在企业内部，都有良好的职业形象和口碑。

  7. 在现代社会中，没有大经济环境和企业的支持，个人的高管梦恐怕很难实现。白领所在企业愿意并有计划帮助员工实现其高管的职业目标的占23%；46%的企业需要先考察员工，再决定是否对其进行培养；还有31%的企业明确表示职业发展是个人的事情，不打算对此负起责任。

  8. 年富力强又经验丰富的年龄，国外高管的年龄普遍在45岁到50岁之间，而中国高管的年龄普遍偏年轻，在40岁左右。通常情况下，一个人24岁大学毕业，从普通员工做到高管的时间是15一 20年左右。

    目前我国在白领为自己制定的高管晋升规划中，46%的人计划用5年时间实现目标；46%的人计划用10年；所需时间最长的是15年，占8%；而计划用15年以上的白领为零。

  9. 万事具备、只欠东风。晋升的资源是机遇，要有选择地加大自己的机遇空间。首先。和上司沟通公司对高管的发展计划，看看自己在一段时期内是不是有发展的机会。其次，如果公司没有机会，那么最好的方式就是跳槽——选择在行业中转换企业的方式，在一家规模相仿的企业中找到总监的职位。45%的人表示要通过有计划的跳槽来实现螺旋式的发展，这是对能力足够，但空间不足的中层经理是最好的方法。但频繁跳槽最容易留下不忠诚的印象，现在企业越来越趋向于认同对职业本身的忠诚，而不是对企业个体的忠诚。
