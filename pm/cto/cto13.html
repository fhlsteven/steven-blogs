# 关于工作邮件的使用建议与说明

为了便于工作使用，使老大们了解工作进程，节约老大们的宝贵时间,特向大家推荐一种发送工作邮件的办法
这方式是经过实践证明很有效的方式，特别是几百人的部门里作用相当突出！希望对大家的工作起到更好推动作用.

# 以下情形大家应尽快让自己主管或是老板知道

  1. 布署的作务完成工作时,应尽快向直属上级汇报工作进程；
  2. 工作进行到一定程度，学会每天汇报一下进度情况,包括难度等.（很适合部门员工使用）
  3. 预料工作会拖延时，要及时向上司汇报。这点很重要,希望各位认真对待!

PS:部门发邮件,请检查一下语法与错别字等问题,特别是对外邮件!

# 1. 部门经理/总监发给上级的邮件,抄送各位组员的邮件

方式: 在邮件主题中加上[B-E]关键字,注:B代表Boss,E代表组员,Employee
这些邮件大家一般都属于较优先级,因为一般都是较重要的事情.大家要认真对待.

1.1 一些较重要制度决定事项,需上级批准或是马上决策的.主题中加上[B-E++]关键字 ++代表相当着急，相当重要。+代表重要，但不紧急
1.2 一些形式上的邮件,不重要的. 主题中加上[B-E--]关键字,上级可看可不看! -(一个减号)代表一般性的事务，不着急，--代表纯流程性的事务，老大们可以不看.

# 2. 因工作汇报等原因,部门经理发给组员的邮件需抄送给上级的邮件

方式:

2.1 在邮件主题中加上[E-FB--]关键字,这些邮件上级可以看或是不看.重要性不高.
2.2 在邮件主题中加上[E-FB++]关键字,让上面知道这是项目进度报告邮件，重要进度报告

3,各位组员发给部门经理/总监/主管 邮件时

方式:一般工作分级别对待,[W1],[W2],[W3],[W4]分别表示相当紧急,紧急,一般,不紧张(资料备份等),注:W代表研发Working意思

# 另外,对工作进度汇报邮件请加以下关键字(可选)

1.如果是项目完成,请在优先权别后面加上[F]关键字,表示任务完成报告 注:Finish
2.如果是项目进度报告,请在优先权别后面加上[P]关键字,表示任务进展报告
